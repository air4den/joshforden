# My Website - Josh Forden

**[joshforden.com](https://joshforden.com)**

Showcasing my interests, experience, skills, and projects. 

## Tech Stack
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- CSS 

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/air4den/josh_forden_portfolio.git
   cd josh_forden_portfolio
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Run the development server:**
   ```sh
   npm run dev
   ```

## Deployment

This site is automatically deployed using GitHub Actions. Just push changes into `main`.
 
To build for production locally:

```sh
npm run build
```
